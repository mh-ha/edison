# edison

**edison:**
- `__pycache__/`
- `configs/`
    - config 관련 폴더입니다.
- `constants/`
    - generation과 관련된 constant 파일을 가지고 있습니다.
- `layers/`
    - 주요 layer class를 저장한 폴더입니다.
- `metrics/`
    - 주요 metric을 계산하는 코드를 가지고 있습니다.
- `modules/`
    - lightning module을 가지고 있습니다.
- `pipes/`
    - modules의 클래스들을 동작하는 코드를 가지고 있습니다.
- `repositories/`
- `resources/`
- `schemas/`
    - diffusion model의 output을 정의한 클래스를 가지고 있습니다.
- `tests/`
- `utils/`
    - 주요 계산 코드들을 모아두었습니다.
